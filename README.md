# 📝 HTML Bootstrap JavaScript Form Validation

A responsive **Registration Form Validation Project** built with **HTML5, Bootstrap 5, and JavaScript**. This project validates user input before submission by checking empty fields, email format, phone number length, and password length using JavaScript.

## ✔️ Validation Rules

### 👤 Name

- Cannot be empty

### 📧 Email

- Cannot be empty
- Must be a valid email format

Example:

```
example@gmail.com
```

---

### 📱 Phone Number

- Cannot be empty
- Must contain exactly **11 digits**

Example:

```
01712345678
```

---

### 🔒 Password

- Cannot be empty
- Must contain at least **7 characters**

Example:

```
password123
```

---

## 📖 How It Works

1. User enters all required information.
2. JavaScript checks whether fields are empty.
3. Email is validated using Regular Expression.
4. Phone number length is checked.
5. Password length is validated.
6. If all inputs are valid, the form can be submitted.
7. Otherwise, an alert message displays the validation error.
